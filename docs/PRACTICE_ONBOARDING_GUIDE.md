# Practice Onboarding Guide

Route: `/provider/onboarding` (owner/admin).

## Steps

1. Create / confirm practice
2. Add locations
3. Invite owner
4. Add staff
5. Add providers
6. Assign roles & location access
7. Configure schedules
8. Configure patient portal
9. Configure reminders
10. Configure Google reviews
11. Import patients
12. Review launch checklist
13. Approve pilot mode

## Progress

- Percent complete from step checklist
- Blockers called out (locations, owner, providers, PHI/pilot)
- Links into existing setup pages (practice setup, team, migration, pilot launch)

- Staff onboarding: `/provider/settings/staff-onboarding`

## Safety

- Demo org cannot enable controlled pilot
- Live PHI remains fail-closed until readiness + BAAs + explicit flags
